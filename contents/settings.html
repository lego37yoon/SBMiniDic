번역 설정

카카오 개발자 페이지에서 이미 등록된 애플리케이션의 REST API 키를 입력해주세요. 2023년 6월 말 주식회사 카카오의 정책에 따라 서비스가 완전히 종료됩니다. 이와 관련하여 대체 서비스인 주식회사 카카오엔터프라이즈의 번역 서비스는 추가가 까다로운 관계로 현재 미정입니다.


네이버 개발자 페이지에서 애플리케이션 등록 시, 다음과 같이 설정해주세요. 미설정시 이용이 어렵습니다.

  • 사용 API: Papago 번역, Papago 언어감지 (자동 감지 선택 시)
  • 비로그인 오픈 API 서비스 환경: Web
  • 웹 서비스 URL: file://localhost

Google Cloud 콘솔 페이지에서 프로젝트 등록 후, 다음 절차를 따라주세요.

  1. API 및 서비스 메뉴로 이동
  2. 사용자 인증 정보 선택
  3. + 사용자 인증 정보 만들기 클릭
  4. API 키 선택

API 키의 이름을 누르고 API 제한사항으로 Cloud Translation API만 사용할 수 있도록 설정하시는 것을 권장합니다.

번역 모드를 사용하려면 API 키가 반드시 필요합니다.
